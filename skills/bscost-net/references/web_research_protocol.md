# Web Research Protocol

## Source policy
1. Prefer primary sources (official repos/docs/papers).
2. Record URL + access date + commit/release tag when possible.
3. Do not use secondary blog claims as final evidence.

## Minimal capture template
1. `source_name`
2. `url`
3. `date_accessed`
4. `version_or_commit`
5. `why_selected`
