# BSPDN Goal Contract

## Target A

- dynamic power reduction about `8%`,
- area/timing non-worse (`delta_area<=0`, `delta_wns>=0`, `delta_tns>=0`).

## Target B

- power/area non-worse,
- frequency uplift `>=5%` validated by period-sweep evidence.

## Selection Rule

Use Goal A by default for dynamic-power reduction campaigns.
Use Goal B only when the task explicitly targets frequency uplift with non-worse power and area.
