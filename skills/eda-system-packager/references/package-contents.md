# Package Contents

Use this reference when deciding what must appear in a standalone package.

## Required Core Content

1. standalone `AGENTS.md`
2. `skills/` tree needed for the packaged system
3. core `docs/knowledge_base`
4. core `docs/tool_registry`
5. selected `scripts/common`

## Mirroring Rule

If portability requires skill-local script mirrors, ensure the bundle records that mapping explicitly rather than relying on hidden repo-relative paths.
