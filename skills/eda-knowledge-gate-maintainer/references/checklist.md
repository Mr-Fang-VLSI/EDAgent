# Interaction Checklist

## Pre-action
1. Open task brief.
2. Run `knowledge_gate.sh`.
3. Query `tool_catalog.py` with task keywords.
4. Record intended reuse/new creation plan.

## Post-action
1. Record what was changed (scripts/docs/jobs).
2. Record artifacts (monitor paths, summary paths, output files).
3. Record unresolved risks/open items.
4. Append one row to maintenance log.

## Minimal log row template
`| <date> | <scope> | <kb docs> | <tool query> | <reuse/new> | <updates> | <open items> |`
