# Source Priority

Use this ranking when collecting papers:

1. Publisher/official DOI landing page.
2. arXiv official page and PDF.
3. OpenReview official forum page.
4. Conference proceedings page (ACM/IEEE/USENIX/etc.).
5. Author-hosted PDF (only when primary source is unavailable).

Rules:
1. Always capture DOI or arXiv ID when available.
2. Prefer sources with stable identifiers over mirror links.
3. Do not mark a paper as "downloaded" unless local PDF exists.
